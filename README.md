Black 3D Ripper is a Project that integrates the most of 3D VUewers capabilities into one fully fledged one.

The Program currently:\
• Supports formats like: OBJ, CSV, RIP,  NR, STL, DAE, FBX, 3DS, MDL, SMD.\
• Has a support for Individual Submesh Selection AND Texture Assigment for them.\
• Textrue Assigment supports: Color Maps, Normal Maps, Specular Maps, Roughness Maps, Metallioc Maps all with UV Map are individually viewable.\
• Realtime Texture Update, any changezs to the texture will be instantly updated in a 3D Viewer.\
• Submesh separation & connection abuility by loose parts.\
• Ability to change Dark<->Light Mode instanly on "M" Key.\
• Ability to Export any loaded Model to OBJ + MTL Instantly with "Enter" Button.\
• Exported models will have their own Copied textures and Converted from mixed Imnage Formats into a Valid PNG Formats.\
• .NET 4.8 With Open Code Nature for Performance and ease of adding the new File Formats.
